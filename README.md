# Mobile Automation Test Assignment

**Стек:** Appium, Java 22, Selenide, Android Studio (Android 11 / API 30), Maven

**Использованные библиотеки:** JUnit 6, AssertJ, Lombok, Log4j, SLF4J

---

## Тестовые сценарии

### 1. VK Видео (`com.vk.vkvideo`)  
- Цель: проверить воспроизведение видео
- Шаги выполнения:
  1. Пропустить авторизацию и рекламу
  2. Открыть первое видео из ленты
  3. Проверить, что видео воспроизводится
- Проверка: обработан положительный и негативный случай
- Ссылка: [VK Видео на Play Store](https://play.google.com/store/apps/details?id=com.vk.vkvideo)  

### 2. Алхимия: Головоломка (`com.ilyin.alchemy`)
- Цель: проверить получение подсказок за просмотр рекламы
- Шаги выполнения:
  1. Нажать **Играть** в главном меню
  2. Открыть панель подсказок
  3. Получить подсказку за просмотр рекламы
  4. Проверить, что количество подсказок увеличилось до 4
- Проверка: обработан положительный и негативный случай
- Ссылка: [Алхимия на Play Store](https://play.google.com/store/apps/details?id=com.ilyin.alchemy)  

---

## Особенности реализации

- Использован **Page Object Model (POM)** с `AppiumFieldDecorator`
- Динамические элементы ищутся перед использованием (`By`) для избежания `StaleElementReferenceException`
- Ожидание элементов через `WebDriverWait`, вместо фиксированных таймеров 
- Логирование шагов теста через **Slf4j**
- Все тесты успешно завершаются при выполнении сценариев

---

## Запуск тестов

1. Запустить **Appium Server** из командой строки через команду `appium`
2. Подключить своё устройство или запустить эмулятор Android 11
3. Собрать и запустить тесты из командной строки через команду `mvn clean install`
